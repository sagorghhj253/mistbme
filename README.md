# Department of Biomedical Engineering - Academic Website

## 🎓 Overview
A modern, responsive academic website for the Department of Biomedical Engineering (Level-4, Term-II, Session 2024-25) featuring elegant design, medical-themed graphics, and comprehensive course information.

## ✨ Key Features

### 1. **Course Materials Access**
- **Direct Google Drive Integration**: Access all course materials, notes, and study resources
- **Link**: [Course Materials Drive](https://drive.google.com/drive/folders/1BwB6LH9x4-Ymq3TPD23Juo9u8T9jkInd)
- **Available on**:
  - CT-Mid Routine page (bottom of subject-wise view)
  - Individual course selection displays
  - Prominent materials section with animated button

### 2. **Medical-Themed Design Elements**
Enhanced background graphics throughout the site:
- **DNA Helix Patterns**: Subtle diagonal patterns representing genetic sequences
- **ECG/Heartbeat Lines**: Repeating medical monitoring patterns
- **Medical Cross Symbols**: Caduceus (⚕) icons as decorative elements
- **Molecular Structures**: Atomic and molecular pattern overlays
- **Plus Signs**: Medical cross patterns in various sections
- **Gradient Overlays**: Biomedical color schemes with warm accent tones

### 3. **Navigation & Pages**
- **Home**: Hero section with statistics and welcome cards
- **CT-Mid Routine**: Examination schedules (all courses + subject-wise)
- **Class Routine**: Complete weekly timetable
- **Software**: Essential engineering tools with download links
- **Gallery**: Visual showcase with hover effects
- **Contact CRs**: Class representative information

### 4. **Interactive Features**
- Smooth page transitions with fade animations
- Hover effects on all interactive elements
- Medical symbol animations on hover
- Gradient backgrounds with pulse animations
- Parallax scrolling on hero section
- Copy-to-clipboard for contact details
- Mobile-responsive hamburger menu

## 📚 Course Materials Integration

The Google Drive link is integrated in multiple locations:

1. **CT-Mid Routine Page** (Subject-wise view):
   - Dedicated "Course Materials & Resources" section at the bottom
   - Large, prominent button with icons
   - Animated hover effects

2. **Individual Course Selection**:
   - When a specific course is selected
   - Displays below the examination schedule
   - Quick access download button

3. **Button Features**:
   - SVG icons (document and arrow)
   - Smooth color transitions on hover
   - Scales and transforms on interaction
   - Opens in new tab for user convenience

- 968px: Tablet and mobile menu activation

## 🚀 Usage

1. **Open the Website**: Launch `index.html` in any modern browser
2. **Navigate**: Use the top navigation menu
3. **Access Materials**: Click "Download All Course Materials" button
4. **View Schedules**: Switch between "All Courses" and "Subject-wise" views
5. **Contact CRs**: Click on contact details to copy to clipboard
